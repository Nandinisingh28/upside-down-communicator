## 🌌 Overview

The **Upside Down Communicator** is a retro-inspired, immersive web application designed for interdimensional communication. Built with a high-fidelity 1980s aesthetic, it allows users to transmit Morse code-encoded messages across thin spots in reality. 

The system features real-time hardware monitoring, a dynamic "Sanity" engine, and a terrifying "Possessed" mode that triggers when the barrier between worlds becomes too thin.

## 🛠️ Core Features

- **📼 Retro CRT Interface**: Custom CSS-driven CRT scanlines, flicker effects, and phosphorus glow for a true 1983 terminal experience.
- **📡 Morse Encoding Engine**: Real-time translation of text into Morse code pulses with synchronized audio and visual waveform displays.
- **📟 Hardware Dashboard**:
    - **Frequency Dial**: Tune the transmitter to the optimal 600Hz frequency.
    - **Sanity Meter**: Monitor your mental stability—transmitting too long drains your sanity.
    - **Temperature/Voltage Gauges**: Manage system heat; overheating forces an emergency vent.
    - **Spectrum Analyzer**: Real-time signal visualization.
- **🏚️ The "Possessed" Mechanic**: If sanity reaches zero, the interface is overtaken by the Upside Down. Users must perform a multi-stage recovery sequence (tuning, power management, and a secret key pattern) to restore reality.
- **🕵️ Hidden Layers**: Toggle hidden communication layers to reveal encrypted location data and metadata.
- **🥚 Easter Eggs**: Secret trigger words that activate special visual effects and system overrides.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vite.dev/)
- **Styling**: Vanilla CSS (Advanced animations, CRT filters)
- **Audio**: Custom Web Audio API wrapper for procedural signal generation
- **State Management**: React Hooks (useState, useEffect, useCallback, useRef)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Nandinisingh28/upside-down-communicator.git
   cd upside-down-communicator
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🎮 How to Use

1. **Initialize Link**: Click through the portal page to establish the interdimensional link.
2. **Transmit**: Type a message in the console and hit `ENTER`.
3. **Monitor**: Keep an eye on the **Sanity Meter**. If it drops too low, the transmission will become corrupted.
4. **Manage Heat**: If the system overheats, use the `VENT SYSTEM` action to cool down.
5. **Recovery**: If the system becomes **POSSESSED**:
    - **Phase 1**: Tune the frequency dial to exactly ~600Hz.
    - **Phase 2**: Maximize the Power Level.
    - **Phase 3**: Input the secret sequence shown on the screen to purge the system.
